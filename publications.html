Auto-Rubric as Reward: From Implicit Preferences to Explicit Multimodal Generative Criteria

Juanxi Tian, Fengyuan Liu, Jiaming Han, Yilei Jiang, Yongliang Wu, Yesheng Liu, Haodong Li, Furong Xu, Wanhua Li

ARR turns implicit visual preferences into explicit multimodal rubrics, then uses Rubric Policy Optimization to improve text-to-image generation and image editing. The work emphasizes inspectable criteria, reusable reward structure, and artifact-backed evaluation.

Key Idea

Convert human visual preference into explicit criteria that can be read, audited, and reused.

Reusable Artifact

Released model artifacts support generation and editing experiments built around rubric rewards.

Method Rubric Policy Optimization
